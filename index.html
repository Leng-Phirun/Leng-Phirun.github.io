# About Me

I am a penetration tester with a passion for breaking into systems to make them stronger. My work focuses on identifying security weaknesses in networks, web applications, and infrastructure before malicious actors can exploit them.

I approach security with an attacker's mindset and a defender's purpose -- every vulnerability found is an opportunity to improve an organization's security posture.

When I'm not testing systems, I enjoy participating in CTF competitions, researching new attack techniques, and contributing to the security community.
