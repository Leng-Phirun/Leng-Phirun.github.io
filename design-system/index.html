Core surfaces, accents, text, and the light-theme override. Includes an alternate "Dim" accent variant for long-form reading.

# Type

Inter for body + display, JetBrains Mono for terminal moments. Full scale from 0.7rem to 3.5rem hero.

# Spacing & Surfaces

Spacing scale, border radii, and the three-tier shadow system (card-hover, btn-glow, float).

# Components

Buttons, chips, cards, pills, timeline, section headers, nav, and the terminal loader — ready-to-copy markup.

Logo lockup, iconography, background treatments, and the optional matrix-rain motif.
